Nested Arrays:

they are Arraies inside Arraies.

example:

const airplaneSeats = [
    ['Ruth', 'Anthony', 'Stevie'],
    ['Amelia', 'Pedro', 'Maya'],
    ['Xavier', 'Ananya', 'Luis'],
    ['Luke', null, 'Deniz'],
    ['Rin', 'Sakura', 'Francisco']
];

--
to call a certen item inside an array
airplaneSeats[3][1] = "Hugo";
